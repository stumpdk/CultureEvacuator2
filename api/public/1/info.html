Starbas, API v. 0.1

Et simpelt API til forespørgsler i Starbas.net

Informationerne i Starbas.net er struktureret i tre hovedgrupper: Papirarkivalier, digitale arkivalier og AV-materiale. Pt. er det kun muligt at forespørge i AV-materiale.

AV-materiale

Service http://www.starbas.net/api/1?type=av

Returnerer en liste med AV-materiale, herunder beskrivelse og arkivnavn.

Parametre:

  • id: Materialets unikke id
  • nummer: Materialets nummer
  • arkiv: Navn på arkivet der ejer materialet.
  • fritekst: Fritekstsøgning i beskrivelse af materialet. Ved søgning på flere udtryk benyttes ";"
  • aar: Materialets år.
Eksempel http://www.starbas.net/api/1/?type=av&fritekst=kildevæld&id=53349
